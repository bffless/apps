# Example Prompts

Reference prompts in each house style. Copy as starting points; adapt headline, subject, and accent details to the current brief.

## retro-blueprint — YouTube thumbnail

```
A 16:9 YouTube thumbnail in the style of a vintage technical manual or developer's design notebook — cream/parchment background (#F1E9D5) with a faint paper grain texture, like an old engineering blueprint reprinted on aged stock.

Top edge: small uppercase metadata strip in muted black, separated by bullets: "BFFLESS • COMMUNITY EDITION • S01 EP01 • DEPLOY". Above the headline, a thin tag reads "OPEN SOURCE / SELF-HOST" in a small monospace font.

Left half: massive headline in heavy black grotesque sans-serif (Inter Black / Söhne Breit weight), stacked on 3 lines:
"DEPLOY
YOUR FIRST
SITE" — with a small bright red rocket icon (🚀 hand-drawn style) tucked next to "YOUR" for a single pop of color. Below it, a smaller pill that reads "BFFLESS" in clean white text on a solid black rectangle. Beneath that, 3-4 lines of body copy in a smaller grotesque (justified): a brief description of the tutorial.

Right half: a hand-drawn technical-blueprint style schematic in dark ink (#1A1A1A) on the parchment background. Wireframe panels showing the system being explained, connected with thin pencil-style arrows and dotted lines, like an engineer sketched the system on a napkin. Add tiny handwritten labels next to each diagram.

Bottom edge: a horizontal row of 4 boxed category labels in small caps, each with a tiny hand-drawn icon: "AUTH", "PROXY", "PIPELINES", "STORAGE". Thin black hairline border around each box, equally spaced.

Style: vintage developer-tool aesthetic — think Linear's brand crossed with a 1970s IBM technical manual or a Braun product brochure. Confident, editorial, slightly nerdy. Hand-drawn schematic feel but with crisp modern type. No photorealism, no 3D renders, no stock illustrations.

Color palette: parchment #F1E9D5 background, ink black #1A1A1A for type and diagrams, single bright red accent #E63946 (used only on the rocket icon), warm off-white highlights. Limit to 3 colors total.

Avoid: photorealistic humans, glowing neon, dark mode/space backgrounds, more than 6 words of large text, gradients, drop shadows, modern "SaaS gradient mesh" look, generic cloud icons, busy backgrounds.
```

## modern-dev-tool — YouTube thumbnail (watch-me-code / live-build)

For "I'm gonna build X in one sitting" videos. Distinguishing markers vs retro-blueprint: dark navy background, code editor mock, `WATCH ME CODE` pill, cyan underline on the proof artifact, "1 sitting" footer.

```
A bold, high-contrast 16:9 YouTube thumbnail for a watch-me-code / live-build developer video. Dark navy flat background #0B1226 with a faint dot-grid texture (small dots in a regular matrix at ~6% opacity, NOT line grid, NOT a radial gradient).

Top-left: a small "WATCH ME CODE" pill — thin rounded rectangle, white text, hairline white-or-cyan border, ~3% of canvas height. This is the unmistakable watch-me-code content-type marker.

Composition: split roughly 55/45 left-to-right. Left holds the headline stack. Right holds a tilted code editor mock.

Left half — headline in heavy white sans-serif (Inter Black / Söhne Breit), stacked on 2-3 lines. The portion of the headline that names the thing being built (the URL, the stack name, the proof artifact) gets a hand-drawn cyan underline #22D3EE — a single confident stroke beneath those words, not a perfect rectangle, slightly rough as if drawn live. Beneath the headline, a small footer line in monospace at ~60% opacity: "github · j5s.dev · 1 sitting" or equivalent bullet-separated tokens. The "1 sitting" / "live" framing is the watch-me-code tell — keep it.

Right half — a stylized macOS-style code editor window, tilted ~6° toward the viewer, traffic-light dots top-left, thin cyan #22D3EE outline. Show a few lines of monospaced syntax-highlighted code (JSX / HTML / TS / shell) relevant to the video's subject. Window has dark chrome matching the background; soft shadow at the base for separation.

Accent details: a few small cyan #22D3EE dots or dashes in the empty corners for negative-space rhythm — sparing, not a particle field. Optional single magenta #D946EF highlight used once (a tick mark, a syntax fragment).

Style: confident, restrained, live-build energy — every element load-bearing. Vercel/Linear/Railway thumbnail family but with the watch-me-code pill and cyan underline as the unique BFFless markers. High contrast for mobile readability. Cinematic lighting, sharp focus, 4K detail.

Color palette: dark navy #0B1226 background, off-white #F8FAFC text, electric cyan #22D3EE for underline + editor outline + corner dots, optional sparing magenta #D946EF. 4 colors maximum.

Avoid: parchment textures, hand-drawn schematic diagrams, "TUTORIAL" or "S01 EP##" tags, body copy paragraphs (those are retro-blueprint tutorial territory), photorealistic humans, generic cloud icons, busy backgrounds, radial gradients, neon glow on the headline itself (only the underline + editor outline glow), more than 5 words per headline line, hexagonal Web3 motifs.
```

## editorial-print — blog hero

```
A 1200×630 blog hero image styled as a magazine cover spread.

Composition: warm off-white background (#FDF6E3), large editorial serif headline (GT Sectra / Tiempos Headline feel) on 2-3 lines, left-aligned, generous margins on all sides. Beneath the headline, a small uppercase kicker line ("ISSUE 04 — INFRASTRUCTURE") in monospace. To the right of the headline, a single restrained illustration: a duotone treatment of an abstract architectural form, deep ink color (#1A1A2E) on the cream.

No secondary illustrations, no icons, no badges. Just the headline, kicker, and one focal mark.

Style: thought-leadership editorial. Restrained, confident, slow. Think Harper's, Aeon, or The Browser. Not a SaaS landing page.

Color palette: warm cream #FDF6E3 background, deep ink #1A1A2E type and illustration, one warm accent #B85042 used only on the kicker text. 3 colors total.

Avoid: gradient mesh, neon, multiple illustrations, drop shadows, icons, glassmorphism, anything that screams "tech startup."
```
